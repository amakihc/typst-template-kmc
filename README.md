# Typstテンプレート
## Typstとは
- TeXの上位互換です。異論は認めません。文法も簡潔、環境構築も簡単、コンパイル爆速。
- [Typst公式サイト](https://typst.app/)
## 説明
- `template.typ`にはテンプレートが記述されています。フォントやデザインを変更したい場合はここを参照してください。
- `main.typ`に本文を記述してください。最低限の例が書かれているので、理解したら消して使用してください。
## 注意点
- もともと公開するつもりがなかったので特に`template.typ`がごちゃごちゃしています。
- バグがあったらすみません。自力で直してください。
